# 24 Curriculum-Aligned Interactive Learning Programmes

This package contains 24 browser-based learning programme prototypes from Preschool to Form 5.

## Package structure

- `index.html` — catalogue of all programmes
- `programs/` — 24 separate HTML programme pages
- `assets/learning-style.css` — shared responsive design
- `assets/programs-data.js` — programme content
- `assets/learning-engine.js` — quizzes, missions, progress and certificate logic

## Features in every programme

- Curriculum-focus overview
- Three learning modules
- Three scenario missions
- Five-question mastery assessment
- Automatic scoring and badge
- Student reflection pad
- Reflection text download
- Printable certificate after a score of at least 60%
- Automatic progress saving with browser localStorage
- Mobile and desktop responsive interface
- No API or database required

## Run locally

Open `index.html` directly, or use VS Code Live Server.

## Upload to a website

Upload the complete folder while preserving the `assets` and `programs` directories.

## Curriculum note

These are thematic curriculum-aligned development prototypes. Before formal deployment,
map and verify each programme against the latest official DSKP, including the
relevant Content Standards, Learning Standards and Performance Standards.
