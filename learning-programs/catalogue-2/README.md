# Learning Quest Catalogue 2

A standalone interactive HTML catalogue containing 24 additional programme
concepts from Preschool to Form 5.

## Programme numbering

The catalogue continues from the earlier collection:

- Programme 49 to Programme 72
- Two programme concepts for every education level
- Preschool, Year 1–6 and Form 1–5

## Features

- Responsive catalogue cards
- Keyword search
- Education-phase filter
- Subject filter
- Detailed programme blueprint modal
- Learning outcomes
- Suggested interactive activities
- Assessment approach
- Proposed student product
- Browser-based shortlist
- Export shortlist as TXT
- No external API or library required

## Run

Open `index.html` in a modern browser.

## Website installation

Upload the entire `learning-programs/catalogue-2` folder to your website, for example:

`/learning-programs/catalogue-2/`

The expected public address would then be similar to:

`https://www.mathtoolshub.com/learning-programs/catalogue-2/`

## Curriculum note

The catalogue is thematically aligned with areas found across current curriculum documentation. Before formal classroom use, map each selected
programme to the exact current DSKP Content Standards, Learning Standards and
Performance Standards.
