Select a cell and choose a number.

How to Play

  1. Fill every empty square with a number from 1 to 9.
  2. Each row must contain every number once.
  3. Each column must contain every number once.
  4. Each 3×3 box must contain every number once.

Keyboard Controls

Use number keys 1–9, Backspace/Delete to erase, and arrow keys to move.

Learning Tip

Start with rows, columns, or boxes that already contain many numbers. Eliminate impossible choices first.
