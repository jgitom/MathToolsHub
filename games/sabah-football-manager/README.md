# Sabah Football Manager

A self-contained browser football management simulator.

## Main systems

- Squad management and Starting XI selection
- Tactical formations and mentality
- Weekly training
- Transfer market
- League fixtures and table
- Live match simulation
- Player fitness, morale, ratings and goals
- Club budget, wages, balance and revenue
- Season progression
- Automatic browser save using localStorage

## Run

Open `index.html` in Chrome, Edge, Safari or Firefox.

Recommended with VS Code:

1. Open the extracted folder.
2. Right-click `index.html`.
3. Select **Open with Live Server**.

## Notes

- The clubs and players are fictional.
- No API, database or internet connection is required.
- A future online version can connect accounts and cloud saves to Supabase.
