Select a cell and choose a number.

How to Play

  1. Fill each row with numbers from 1 to the board size.
  2. Do not repeat a number in any row or column.
  3. Numbers inside each bold cage must produce the target.

Cage Symbols

+ Add
− Subtract
× Multiply
÷ Divide

Learning Tip

Start with single-cell cages, then solve cages with only one possible arithmetic combination.

Keyboard

Use number keys, arrow keys, and Backspace/Delete to erase.
