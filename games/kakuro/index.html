Select a white cell and choose a number.

How to Play

  1. Fill white cells with digits from 1 to 9.
  2. The digits must add up to the clue shown.
  3. Do not repeat a digit within the same run.
  4. Each horizontal and vertical run has its own clue.

Clue Direction

Upper-right number = vertical sum
Lower-left number = horizontal sum

Learning Tip

Start with short runs and small sums. For example, a two-cell sum of 3 must be 1 and 2.

Keyboard

Use number keys 1–9, arrow keys, and Backspace/Delete to erase.
