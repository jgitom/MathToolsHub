Math Learning Games 31–40

HOW TO USE
1. Extract every file into the same folder.
2. Open index.html in Chrome, Edge, Firefox, or Safari.
3. To publish, upload the complete folder including:
   - game_style.css
   - game_engine.js
   - all 10 HTML files
   - index.html
4. The package works offline and does not require an API.

IMPORTANT
The individual HTML games use the shared game_style.css and game_engine.js files.
Keep all files together in the same directory.
