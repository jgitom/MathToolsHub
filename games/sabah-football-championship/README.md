# Sabah Football Championship

A self-contained HTML5 football game that runs in a modern browser.

## Features

- Keyboard controls: WASD / arrow keys
- Shoot with Spacebar
- Mobile touch controls
- AI opponent and goalkeepers
- Match timer and scoring
- Multiple teams
- Three difficulty levels
- Level progression
- Coins and three free matches
- Progress saved using browser localStorage
- No external images, libraries or API required

## Run

Open `index.html` in a browser.

For best results, use VS Code Live Server:

1. Open the project folder in VS Code.
2. Right-click `index.html`.
3. Choose **Open with Live Server**.

## Controls

- Move: WASD or arrow keys
- Shoot: Spacebar
- Mobile: on-screen directional pad and Shoot button
